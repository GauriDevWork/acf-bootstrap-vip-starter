== Description ==

TechArk Theme Builder is a powerful tool that helps developers quickly generate custom WordPress starter themes with built-in Redux Framework integration. Perfect for creating client sites or starting new theme development projects.

= Features =

* **Custom Theme Generation** - Create themes with your custom name, prefix, and text domain
* **Redux Framework Integration** - Pre-configured Redux options for typography, colors, header, footer, and more
* **Template Creator** - Built-in page template creation tool
* **Global Design Settings** - Pre-configured options for buttons, 404 pages, blog layouts
* **One-Click Download** - Generate and download ready-to-use theme ZIP files
* **WordPress.org Compliant** - Generated themes follow WordPress coding standards
* **Developer Friendly** - Clean, well-documented code with proper prefixing

= Generated Theme Includes =

* Redux Framework integration with custom prefix
* Typography settings (Content & Heading fonts)
* Color scheme settings (Primary, Secondary, Links)
* Button styling options
* Header builder with menu settings
* Footer builder with widget areas
* Blog and post layout options
* 404 page customization
* Template creator functionality
* Responsive design starter files

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/techark-theme-builder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Tools > TechArk Theme Builder to start generating themes.

== Frequently Asked Questions ==

= Do I need Redux Framework installed? =

No, the generated themes include instructions for installing Redux Framework via TGM Plugin Activation.

= Can I customize the generated themes? =

Yes! The generated themes are fully customizable. They serve as a starting point with best practices already implemented.

= What PHP version is required? =

PHP 7.4 or higher is required for both the plugin and generated themes.

= Is the generated theme compatible with WordPress.org? =

Yes, generated themes follow WordPress coding standards and best practices.
