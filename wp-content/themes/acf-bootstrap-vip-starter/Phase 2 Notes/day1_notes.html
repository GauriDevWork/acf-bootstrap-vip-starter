Key decisions made today

ACF options page via UI, not code. ACF PRO 6.x handles this natively — no acf_add_options_page() needed.

Use wp_strip_all_tags() not wp_kses() for CSS. wp_kses() strips valid CSS syntax — wrong tool for this job.

Branch naming convention confirmed: feat/ fix/ perf/ docs/ chore/ prefix on all branches.
